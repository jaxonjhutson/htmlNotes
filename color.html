There are three ways to do color.

- Name
- Hex
- Styles using RGBA (doesn't work in HTML, use CSS)

Name

- Name is limited to 140ish defined names that you can look up

Hex (Hexadecimal)

- If you use hex, you have 16 million and change, which is a pretty big deal (though there are only 216 web safe colors).
- Hexadecimal is a base 16 numbering system, which means that it has 16 digits
- Those digits being 1, 2, 3, 4, 5, 6, 7, 8, 9, 0, A, B, C, D, E, and F.
- Browsers, however, interpret this differently, so to make web safe colors you must have sets of the same digit (ex. 11, AA, or FF).
- As for syntax, you start with a # then follow with six digits.

RGBA

- RGBA stands for red, green, blue, and alpha
- Can only be used with stylesheets.
- Syntax will be: rgba(0-255, 0-255, 0-255, 0-1)
- Alpha basically controls opacity
